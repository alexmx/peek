# Peek Testing Scenarios

Realistic testing scenarios using Xcode and iOS Simulator.

## Xcode Navigation

1. Find and click the Run button in Xcode
2. Switch to the Issue Navigator tab in Xcode
3. Open the File menu in Xcode and click 'New File...'
4. Find all disabled buttons in Xcode's toolbar
5. What file is currently open in Xcode's editor?
6. Click the debug area toggle button in Xcode
7. Find the search field in Xcode's navigator and type a filename

## Xcode Build & Debug

8. Check if Xcode shows any build errors in the issue navigator
9. Take a screenshot of Xcode's debug console output
10. Find the Stop button and check if it's enabled (i.e., is something running?)
11. Read the status bar text in Xcode to see if the build succeeded
12. Click Product > Clean Build Folder in Xcode

## iOS Simulator

13. List all Simulator windows and take a screenshot of the active one
14. Find and click the Home button in the iOS Simulator
15. What app is currently visible in the Simulator? Read the navigation bar title
16. Take a screenshot of just the Simulator screen, cropped to exclude the chrome
17. Find all buttons visible in the Simulator app
18. Click on a specific table row in the Simulator
19. Type 'hello world' into a text field in the Simulator
20. Find the back button in the Simulator's navigation bar and tap it

## Cross-App Workflows

21. Switch from Simulator to Xcode, click Run, then switch back to Simulator and take a screenshot after 5 seconds
22. Take screenshots of both Xcode and the Simulator side by side
23. Find the Simulator device name from its window title and the Xcode scheme name from its toolbar
24. Activate the Simulator, find a button labeled 'Login', click it, then take a screenshot of the result

## Monitoring & Debugging

25. Watch Xcode for any value changes while I trigger a build
26. Monitor the Simulator for UI changes while I tap through screens
27. Inspect the full accessibility tree of the Simulator at depth 5 and find all interactive elements
28. Find all StaticText elements in the Simulator to read the screen content

## Menu Bar

29. Show me Xcode's Debug menu structure
30. Click 'Simulate Memory Warning' from Xcode's Debug menu while the Simulator is running
31. Find the keyboard shortcut for 'Run' in Xcode's Product menu
